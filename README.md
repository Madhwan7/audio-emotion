# 🎧 Audio Emotion Detection using Machine Learning

## 📌 Overview
This project focuses on detecting human emotions from audio (speech) using machine learning techniques. It analyzes voice features such as tone, pitch, and frequency to classify emotions like happy, sad, angry, and neutral.

---

## 🚀 Features
- Detect emotions from audio files
- Feature extraction from speech signals
- Machine learning-based classification
- Supports multiple emotion classes
- Visualization of audio signals

---

## 🛠️ Tech Stack
- Python
- Librosa
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
